## Installation

1. Create .env file
2. Set vars:
```bash
PGHOST=
PGPASSWORD=
PGDATABASE=
PGUSER=
PGPORT=
```
3. Install dependency
```bash
$ yarn
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# test coverage
$ npm run test:cov
```
